# Expert Booking System

A minimal expert booking application (frontend + backend) that lets users browse experts, register experts, and book sessions.

Live demo (deploy): https://expert-booking-system-murex-delta.vercel.app/register-expert

Repository: https://github.com/JaiSinghRajput/expert-booking-system

## Quick Start

Prerequisites:
- Node.js (v16+ recommended)
- pnpm (or npm/yarn)
- MongoDB (Atlas URI if you want persistence)

Run the server:

```
cd server
pnpm install
# create .env with MONGO_URI and CLIENT_URL as needed
pnpm start
```

Run the client:

```
cd client
pnpm install
pnpm dev
```

Open the app in your browser and visit the expert registration page:

https://expert-booking-system-murex-delta.vercel.app/register-expert

## API Notes
- POST `/experts` — register a new expert (used by the registration page)
- GET `/bookings?email=you@example.com` — list bookings for an email (responses include `expertName`)

## Contributing
PRs welcome. For local development, ensure `MONGO_URI` is set in `server/.env` to persist bookings between restarts.
